## Overview

This is the documentation site for a Proxmox VE home lab environment running
Docker-based services across AI/ML, media, gaming, blockchain, and development
workloads.

## Services

- **AI Stack** — Ollama, Open WebUI, Kokoro TTS, ComfyUI
- **Development** — Coder cloud workspaces
- **Deployment** — Dokploy self-hosted PaaS
- **Media** — Jellyfin with automated downloading
- **Gaming** — Sunshine + Steam cloud gaming
- **Blockchain** — LUKSO node with monitoring
- **Networking** — WireGuard VPN tunnels
