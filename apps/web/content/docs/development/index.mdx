## Overview

[Coder](https://coder.com) provides cloud development environments that run in Docker containers on the home lab. Each workspace is a full Linux environment with VS Code, Node.js, Docker, and blockchain development tools — accessible from any device via a browser.

## Why Coder?

- **Remote access**: Develop from any device with a browser — laptop, tablet, or another machine
- **Context-specific workspaces**: Spin up isolated environments per project
- **Easy reset**: Destroy and recreate workspaces without losing your host setup
- **Pre-configured tools**: PNPM, Node.js, Foundry, Oh My Zsh, and 19 VS Code extensions installed automatically

## Quick start

```bash
# Install Coder
curl -L https://coder.com/install.sh | sh

# Push the template
cd infra/coder/template
coder templates push web3-dev-workspace

# Create a workspace
coder create --template web3-dev-workspace my-workspace

# SSH in
coder ssh my-workspace
```
