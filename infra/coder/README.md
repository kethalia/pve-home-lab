# coder - Cloud Development Environment

Develop remotely from any device.
Have context specific workspaces.
Easy workspace reset, with automatic install of packages like pnpm, nvm, postgres, etc.

## Dependencies

### Docker

You need Docker to deploy [this template](./template/README.md).
Check [this guide](../docker/README.md) to deploy docker.

## Install Coder

Use the following command to get coder installed. For more information check [this](https://coder.com/docs/install).

```shell
curl -L https://coder.com/install.sh | sh
```
